# Projeto CRUD 

Este projeto é um sistema de CRUD (Create, Read, Update e Delete) desenvolvido como parte dos meus estudos em **Análise e Desenvolvimento de Sistemas**. O objetivo principal foi aplicar conceitos de lógica de programação, manipulação de dados e versionamento de código com Git.

## 🚀 Funcionalidades
- **Cadastrar**: Inclusão de novos registros no sistema.
- **Visualizar**: Listagem de todos os itens armazenados.
- **Atualizar**: Alteração de informações de registros existentes.
- **Excluir**: Remoção de itens com segurança.

## 🛠 Tecnologias Utilizadas
- **Linguagem:** C#
- **Plataforma:** .NET 10
- **Ferramenta de Versionamento:** Git & GitHub
- **IDE:** Visual Studio / VS Code

## 📁 Estrutura do Projeto
O projeto foi estruturado buscando manter o código limpo e organizado:
- `Program.cs`: Ponto de entrada da aplicação e lógica principal.
- `.gitignore`: Configurado para ignorar arquivos de compilação (`bin/` e `obj/`), seguindo as boas práticas de desenvolvimento.
